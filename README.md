# coran 
Découvrez une application complète dédiée au Saint Coran, conçue pour vous accompagner dans votre apprentissage au quotidien. Vous y trouverez toutes les sourates du Coran, accessibles facilement, afin de pouvoir lire, apprendre et mémoriser à votre rythme.

L’application vous permet de parcourir l’ensemble des sourates, de suivre votre progression et d’améliorer votre compréhension grâce à une présentation claire et simple. Que vous soyez débutant ou déjà avancé, cette application est idéale pour approfondir vos connaissances et renforcer votre apprentissage du Coran chaque jour.
