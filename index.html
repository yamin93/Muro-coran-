Le Saint Coran

Score :

Fiche : Les Ablutions (Wudu)

1. Intention : Dire "Bismillah".

2. Mains : Laver 3 fois jusqu'aux poignets.

3. Bouche & Nez : Rincer 3 fois chaque.

4. Visage : Laver 3 fois en entier.

5. Bras : Laver jusqu'aux coudes 3 fois (droite puis gauche).

6. Tête & Oreilles : Passer les mains mouillées 1 fois.

7. Pieds : Laver jusqu'aux chevilles 3 fois.
