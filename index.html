🚿 Les Ablutions (Wudu)

1. L'intention : Commencer par "Bismillah".

2. Les mains : Laver 3 fois jusqu'aux poignets.

3. La bouche : Rincer 3 fois avec la main droite.

4. Le nez : Aspirer l'eau et rejeter 3 fois.

5. Le visage : Laver 3 fois (du front au menton).

6. Les bras : Laver jusqu'aux coudes 3 fois (droite puis gauche).

7. La tête : Passer les mains mouillées sur les cheveux et les oreilles 1 fois.

8. Les pieds : Laver jusqu'aux chevilles 3 fois (droite puis gauche).

📖 Apprendre le Coran

1. L'alphabet : Commencer par apprendre les lettres arabes (Al-Qaïda An-Norania).

2. L'écoute : Écouter un récitateur précis (ex: Al-Housary) pour la prononciation.

3. La répétition : Répéter un verset plusieurs fois jusqu'à le connaître par cœur avant de passer au suivant.

4. La régularité : Mieux vaut apprendre 2 versets par jour tous les jours qu'une page entière une fois par mois.

5. Compréhension : Lire la traduction en français pour savoir ce que l'on récite.
